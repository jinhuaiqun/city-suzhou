# 苏州城市研究报告

> 一个 36 岁北京程序员为搬迁苏州所做的系统性调研。
> 覆盖从经济产业到就业、从住房到医疗、从交通到生活成本的全部维度。

---

## 项目背景

本项目的发起者是北京的一名 36 岁程序员，正在评估从北京搬迁到苏州的可行性。报告从多个维度收集数据和信息，最终目标是回答一个核心问题：

**"去苏州，我能过得更好吗？"**

短期答案是：**是的。** 长答案请读下面各篇报告。
