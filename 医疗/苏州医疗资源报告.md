## 四、医学教育与科研

### 高等教育机构

| 机构 | 定位与特色 |
|------|-----------|
| **苏州大学苏州医学院** | 临床医学五年制、"5+3"一体化、医学影像学、口腔医学；临床医学一级学科博士点；放射医学全国领先 |
| **苏州大学儿科临床医学院** | 省内重要儿科人才培养基地，儿科学硕博点，累计培养约 800 名本科儿科医生 |
| **南京医科大学姑苏学院** | 苏州市政府与南京医科大学共建，聚焦研究生教育与临床/公卫研究 |

### 住院医师规范化培训

- 苏州市立医院：国家级住培基地（23 个专业基地）
- 苏大附属儿童医院：国家级住培基地（5 个住培专业基地 + 2 个专培基地）
- 全科、儿科均获国家"重点专业基地"称号

## 七、总结

| 维度 | 评价 |
|------|------|
| 三甲数量 | 16 家，地级市中名列前茅 |
| 顶级水平 | 苏大附一院全国第 29 位，血液病学全国领先 |
| 专科特色 | 儿童、感染病、职业病等专科齐全 |
| 医学教育 | 从本科-硕博-住培的完整链条 |
| 人才政策 | 投入力度大，领军人才可达千万级 |
| 空间布局 | 古城密集，新城加速配套 |
| 产业协同 | 生物医药"一号产业"与临床医疗互相赋能 |

> **核心特征**：苏州医疗体系以苏大附一院为龙头、16 家三甲为骨干、完整医学教育链为支撑，正从"地级市顶级"向"省会级"医疗能级跃升。
